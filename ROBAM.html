Google Sheets Setup Guide
Step 1
Create your Google Sheet
Go to sheets.google.com and create a new blank spreadsheet. Name it ROBAM.
Step 2
Open Apps Script
In your sheet, click Extensions → Apps Script. Delete any existing code. Paste the entire contents of Code.gs (included in the ZIP package). Click 💾 Save.
Step 3
Run Setup Function
In Apps Script, select setup from the function dropdown and click ▶ Run. Accept all permissions when asked. This creates all required sheets automatically.
Step 4
Deploy as Web App
Click Deploy → New Deployment → Type: Web App → Who has access: Anyone → click Deploy. Copy the Web App URL.
Step 5
Connect the App
Paste your Web App URL below and tap Connect. The app will load your data automatically on every open.
